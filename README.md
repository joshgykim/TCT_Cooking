# The California Times Cooking

[The California Times Cooking](https://tct-cooking.herokuapp.com/) is a clone of [The New York Times Cooking](https://cooking.nytimes.com), a digital cookbook platform and cooking guide alike, that helps home cooks discover, save and organize recipes written by its editors. Users can also rate and comment while browsing recipes or curated collections of recipes.
